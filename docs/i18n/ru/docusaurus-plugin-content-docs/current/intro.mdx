# Вопросы и ответы для собеседования по PHP

Добро пожаловать в полную коллекцию вопросов и ответов для собеседования по PHP.

## Структура

Вопросы организованы в 22 основные категории, которые доступны в навигационном меню слева.

- **Категории:** 22 основных раздела (основы PHP, ООП, Laravel и др.)
- **Уровни:** Каждый вопрос отмечен соответствующим уровнем (Junior, Middle, Senior или Expert).
- **Подробные ответы:** Развёрнутые руководства по сложным темам, доступные по ссылке «Answers» в меню.

## Как пользоваться

Выберите категорию в боковом меню, чтобы увидеть все связанные вопросы. Каждая страница содержит вопросы всех уровней сложности по конкретной теме.
