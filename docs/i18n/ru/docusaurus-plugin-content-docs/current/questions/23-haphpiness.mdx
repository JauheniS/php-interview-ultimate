### Что такое именованные аргументы (Named Arguments)?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Именованные аргументы (появились в PHP 8.0) позволяют передавать значения по имени параметра, а не по позиции.
[См. Возможности PHP 8.0](../questions/06-php-7-8-new-features.mdx#что-такое-именованные-аргументы-named-arguments-появившиеся-в-php-80)

### Что такое Union Types и Intersection Types?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Union Types (8.0) позволяют указывать несколько типов для параметра; Intersection Types (8.1) требуют соответствия значения сразу нескольким ограничениям типов.
