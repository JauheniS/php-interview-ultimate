# Официальные пакеты и плагины Laravel

Экосистема Laravel — одна из её сильнейших сторон, предоставляющая собственные пакеты практически для каждой типовой задачи веб-разработки.

## Аутентификация и авторизация

### [Breeze](https://laravel.com/docs/11.x/starter-kits#laravel-breeze)

Минимальная, простая реализация всех функций аутентификации Laravel, включая вход, регистрацию, сброс пароля, подтверждение email и подтверждение пароля. Использует Blade или Inertia.

### [Jetstream](https://jetstream.laravel.com)

Более продвинутый стартовый набор для Laravel. Предоставляет реализацию входа, регистрации, подтверждения email, двухфакторной аутентификации, управления сессиями, поддержки API через Laravel Sanctum и опционального управления командами.

### [Sanctum](https://laravel.com/docs/11.x/sanctum)

Предоставляет легковесную систему аутентификации для SPA (Single Page Applications), мобильных приложений и простых токен-ориентированных API.

### [Socialite](https://laravel.com/docs/11.x/socialite)

Выразительный, удобный интерфейс для OAuth-аутентификации через Facebook, Twitter, Google, LinkedIn, GitHub, GitLab и Bitbucket.

## Производительность и масштабируемость

### [Horizon](https://laravel.com/docs/11.x/horizon)

Horizon предоставляет красивую панель управления и конфигурацию на основе кода для ваших Redis-очередей Laravel. Позволяет легко отслеживать ключевые метрики системы очередей: пропускную способность, время выполнения и сбои задач.

### [Octane](https://laravel.com/docs/11.x/octane)

Многократно ускоряет производительность приложения, обслуживая его с помощью высокопроизводительных серверов приложений, включая Swoole и RoadRunner. Octane загружает приложение один раз, хранит его в памяти и затем обрабатывает запросы на сверхвысокой скорости.

### [Reverb](https://laravel.com/docs/11.x/reverb)

Собственный высокопроизводительный WebSocket-сервер для приложений Laravel, обеспечивающий взаимодействие в реальном времени между сервером и клиентом.

## Мониторинг и отладка

### [Telescope](https://laravel.com/docs/11.x/telescope)

Элегантный инструмент отладки для фреймворка Laravel. Telescope предоставляет обзор входящих запросов, исключений, записей логов, запросов к базе данных, задач в очередях, почты, уведомлений, операций кэширования, планировщика задач, дампов переменных и многого другого.

### [Pulse](https://laravel.com/docs/11.x/pulse)

Инструмент мониторинга состояния и производительности приложения Laravel в реальном времени. Позволяет выявлять узкие места — медленные маршруты и задачи — и отслеживать использование ресурсов сервера.

## Инструменты разработки

### [Sail](https://laravel.com/docs/11.x/sail)

Легковесный CLI для взаимодействия с Docker-окружением разработки Laravel по умолчанию. Sail предоставляет отличную отправную точку для создания приложения на Laravel с использованием PHP, MySQL и Redis без необходимости предварительного опыта работы с Docker.

### [Pint](https://laravel.com/docs/11.x/pint)

Бескомпромиссный инструмент для форматирования PHP-кода. Pint построен на базе PHP-CS-Fixer и обеспечивает чистый и единообразный стиль кода.

### [Dusk](https://laravel.com/docs/11.x/dusk)

Предоставляет выразительный, удобный API для автоматизации и тестирования браузера. По умолчанию Dusk не требует установки JDK или Selenium на локальной машине.

## Другие важные пакеты

### [Cashier](https://laravel.com/docs/11.x/cashier)

Предоставляет выразительный, удобный интерфейс к сервисам подписок Stripe и Paddle. Берёт на себя практически весь шаблонный код для работы с подписками.

### [Scout](https://laravel.com/docs/11.x/scout)

Предоставляет простое, драйвер-ориентированное решение для добавления полнотекстового поиска к моделям Eloquent.

### [Pennant](https://laravel.com/docs/11.x/pennant)

Простая, легковесная библиотека для управления feature flags в приложении.

### [Folio](https://laravel.com/docs/11.x/folio)

Мощный маршрутизатор на основе страниц, упрощающий маршрутизацию в приложениях Laravel.

### [Prompts](https://laravel.com/docs/11.x/prompts)

PHP-пакет для создания красивых и удобных форм в консольных приложениях.
