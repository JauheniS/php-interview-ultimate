# PHP Interview Questions

Welcome to the ultimate repository of PHP interview questions and answers.

## Features

- **130+ Modular Questions**: Each question is its own independent module.
- **Categorized by Level**: Junior, Middle, Senior, and Expert.
- **Detailed Answers**: In-depth explanations for complex topics.
- **Beautiful UI**: Built with Docusaurus and Tailwind CSS.

## How to use

Use the sidebar on the left to navigate through categories. Each category contains questions for different experience levels.
