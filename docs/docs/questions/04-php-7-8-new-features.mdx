# 4. PHP 7/8+ New Features

### What are the main differences between PHP 5.x and PHP 7/8?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** 
The transition from **PHP 5** to **PHP 7/8** brought significant improvements in performance, security, and developer experience.

**Key Improvements in PHP 7/8:**
- **Performance:** PHP 7 is up to twice as fast as PHP 5.6, and PHP 8 introduced JIT (Just-In-Time) compilation for further speed gains in certain workloads.
- **Scalar Type Declarations:** PHP 7/8 supports strict typing for `int`, `float`, `bool`, and `string`.
- **Return Type Declarations:** You can now specify the return type of a function.
- **Null Coalescing Operator (`??`):** Provides a shorthand for checking if a value is set and not null.
- **Spaceship Operator (`<=>`):** Used for combined comparisons (less than, equal to, or greater than).
- **Anonymous Classes:** Support for creating classes on the fly.
- **Improved Error Handling:** Many fatal errors were converted to exceptions in PHP 7, and PHP 8 introduced the `Error` class hierarchy.
- **Match Expressions (PHP 8):** A more powerful and strict version of `switch`.
- **Named Arguments (PHP 8):** Allows passing arguments by name.
- **Property Hooks (PHP 8.4):** Support for computed properties with `get` and `set` hooks.
- **Asymmetric Visibility (PHP 8.4):** Different visibility for reading and writing properties.
- **New Array Functions (PHP 8.4):** `array_find()`, `array_find_key()`, `array_any()`, and `array_all()`.
- **Simplified Class Instantiation (PHP 8.4):** Create class and call method/property without extra parentheses: `new User()->name`.
- **Pipe Operator (PHP 8.5):** New operator (`|>`) for chaining function calls.
- **URI Extension (PHP 8.5):** A new built-in extension for URI parsing and manipulation.
- **Clone With (PHP 8.5):** Modify properties during cloning: `clone $u with { name: 'New' }`.
- **#[\NoDiscard] Attribute (PHP 8.5):** Warn when return value of a function is not used.
- **PIE (PHP 8.5+):** Official PHP Installer for Extensions replaces PECL.

### What are Union Types (introduced in PHP 8.0)?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** Union types allow you to specify multiple types for a parameter, property, or return value, instead of just one.
```php
function sum(int|float $a, int|float $b): int|float {
    return $a + $b;
}
```

---

### What is the Nullsafe Operator (introduced in PHP 8.0)?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** The `?->` operator allows you to safely access properties or methods on a potentially null value without throwing an error.
```php
$country = $session?->user?->getAddress()?->country;
```

---

### What are Enums (introduced in PHP 8.1)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** Enumerations allow you to define a set of named constants, providing type safety for fixed sets of values.
```php
enum Status: string {
    case Pending = 'pending';
    case Active = 'active';
}
```

---

### What is Constructor Property Promotion (introduced in PHP 8.0)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** A shorthand syntax to define and initialize class properties directly in the constructor.
```php
class User {
    public function __construct(public string $name, public int $age) {}
}
```

---

### What are Readonly Classes (introduced in PHP 8.2)?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** Marking a class as `readonly` makes all its properties readonly and prevents dynamic properties.
```php
readonly class Post {
    public function __construct(public string $title) {}
}
```

---

### What are Property Hooks (introduced in PHP 8.4)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** Property hooks allow defining custom logic for getting and setting class properties directly on the property declaration, reducing the need for explicit getter and setter methods.
```php
class User {
    public string $name {
        get => strtoupper($this->name);
        set => strtolower($value);
    }
}
```

---

### What is Asymmetric Visibility (introduced in PHP 8.4)?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** It allows setting different visibility levels for reading and writing a property. This is typically used to have a property that is publicly readable but can only be modified from within the class (private set).
```php
class User {
    public private(set) string $id;
}
```

---

### What are the new array functions introduced in PHP 8.4?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** PHP 8.4 introduced four new functions for searching and validating array elements using a callback: `array_find()`, `array_find_key()`, `array_any()`, and `array_all()`.
```php
$firstEven = array_find([1, 2, 3, 4], fn($v) => $v % 2 === 0); // 2
$anyEven = array_any([1, 3, 5], fn($v) => $v % 2 === 0); // false
```

---

### What is the new class instantiation syntax in PHP 8.4?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** PHP 8.4 allows accessing properties and methods of a newly instantiated class without needing to wrap the `new` expression in parentheses.
```php
// Before PHP 8.4
$name = (new User())->getName();

// PHP 8.4
$name = new User()->getName();
```

---

### What is the Pipe Operator (introduced in PHP 8.5)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** The pipe operator (`|>`) allows passing the result of an expression as the first argument to a function or method, making it easier to chain multiple transformations.
```php
$result = "  hello world  "
    |> 'trim'
    |> 'strtoupper'
    |> 'str_replace'('WORLD', 'PHP 8.5', ...); // "HELLO PHP 8.5"
```

---

### What is "Clone With" in PHP 8.5?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** It allows modifying properties of a cloned object during the cloning process using a syntax similar to object initializers.
```php
$user2 = clone $user1 with {
    name: "New Name",
    email: "new@example.com"
};
```

---

### What is the `#[\NoDiscard]` attribute (PHP 8.5)?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** It's a new built-in attribute that indicates the return value of a function or method must be used. Static analysis tools and some IDEs will warn if the result of such a function is ignored.
```php
#[\NoDiscard]
function calculateImportantValue(): int {
    return 42;
}

calculateImportantValue(); // Static analysis warning: Return value ignored.
```

---

### What is PIE and how it relates to PECL?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** **PIE (PHP Installer for Extensions)** is the modern, official tool for installing PHP extensions, designed to replace the legacy **PECL** (which is now deprecated). It's distributed as a PHAR (like Composer) and works similarly, using PHP Foundation's repository.
```bash
# Example usage
pie install redis
```
