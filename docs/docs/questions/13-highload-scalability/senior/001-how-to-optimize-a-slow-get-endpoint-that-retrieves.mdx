# 118. How to optimize a slow GET endpoint that retrieves many records?

**Answer:** 
- Use pagination.
- Use caching (Redis/Memcached).
- Optimize database queries (indexing, select only needed columns).
- Use Eager Loading to avoid the N+1 problem.
