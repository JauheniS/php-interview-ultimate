# 3. PHP 7/8+ New Features

### What are the main differences between PHP 5.x and PHP 7/8?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** 
The transition from **PHP 5** to **PHP 7/8** brought significant improvements in performance, security, and developer experience.

**Key Improvements in PHP 7/8:**
- **Performance:** PHP 7 is up to twice as fast as PHP 5.6, and PHP 8 introduced JIT (Just-In-Time) compilation for further speed gains in certain workloads.
- **Scalar Type Declarations:** PHP 7/8 supports strict typing for `int`, `float`, `bool`, and `string`.
- **Return Type Declarations:** You can now specify the return type of a function.
- **Null Coalescing Operator (`??`):** Provides a shorthand for checking if a value is set and not null.
- **Spaceship Operator (`<=>`):** Used for combined comparisons (less than, equal to, or greater than).
- **Anonymous Classes:** Support for creating classes on the fly.
- **Improved Error Handling:** Many fatal errors were converted to exceptions in PHP 7, and PHP 8 introduced the `Error` class hierarchy.
- **Match Expressions (PHP 8):** A more powerful and strict version of `switch`.
- **Named Arguments (PHP 8):** Allows passing arguments by name.

### What are Union Types (introduced in PHP 8.0)?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** Union types allow you to specify multiple types for a parameter, property, or return value, instead of just one.
```php
function sum(int|float $a, int|float $b): int|float {
    return $a + $b;
}
```

---

### What is the Nullsafe Operator (introduced in PHP 8.0)?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Answer:** The `?->` operator allows you to safely access properties or methods on a potentially null value without throwing an error.
```php
$country = $session?->user?->getAddress()?->country;
```

---

### What are Enums (introduced in PHP 8.1)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** Enumerations allow you to define a set of named constants, providing type safety for fixed sets of values.
```php
enum Status: string {
    case Pending = 'pending';
    case Active = 'active';
}
```

---

### What is Constructor Property Promotion (introduced in PHP 8.0)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Answer:** A shorthand syntax to define and initialize class properties directly in the constructor.
```php
class User {
    public function __construct(public string $name, public int $age) {}
}
```

---

### What are Readonly Classes (introduced in PHP 8.2)?

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Answer:** Marking a class as `readonly` makes all its properties readonly and prevents dynamic properties.
```php
readonly class Post {
    public function __construct(public string $title) {}
}
```
