# 46. What is the difference between `mysqli` and `PDO`?

**Answer:** 
- `mysqli` is specific to MySQL, while `PDO` supports multiple database systems (PostgreSQL, SQLite, etc.).
- `PDO` supports named placeholders in prepared statements.
