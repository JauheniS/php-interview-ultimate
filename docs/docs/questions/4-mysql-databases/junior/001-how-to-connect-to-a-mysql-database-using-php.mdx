# 45. How to connect to a MySQL database using PHP?

**Answer:** You can use the `mysqli` extension or `PDO` (PHP Data Objects). 
```php
// PDO Example
$pdo = new PDO('mysql:host=localhost;dbname=test', 'user', 'pass');
```
