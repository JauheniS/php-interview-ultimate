# Official Laravel Packages & Plugins

Laravel's ecosystem is one of its greatest strengths, providing first-party packages for almost every common web development task.

## Authentication & Authorization

### [Breeze](https://laravel.com/docs/11.x/starter-kits#laravel-breeze)
A minimal, simple implementation of all of Laravel's authentication features, including login, registration, password reset, email verification, and password confirmation. It uses Blade or Inertia.

### [Jetstream](https://jetstream.laravel.com)
A more advanced starter kit for Laravel. It provides the implementation for login, registration, email verification, two-factor authentication, session management, API support via Laravel Sanctum, and optional team management.

### [Sanctum](https://laravel.com/docs/11.x/sanctum)
Provides a featherweight authentication system for SPAs (Single Page Applications), mobile applications, and simple, token-based APIs.

### [Socialite](https://laravel.com/docs/11.x/socialite)
An expressive, fluent interface to OAuth authentication with Facebook, Twitter, Google, LinkedIn, GitHub, GitLab, and Bitbucket.

## Performance & Scalability

### [Horizon](https://laravel.com/docs/11.x/horizon)
Horizon provides a beautiful dashboard and code-driven configuration for your Laravel-powered Redis queues. It allows you to easily monitor key metrics of your queue system such as job throughput, runtime, and job failures.

### [Octane](https://laravel.com/docs/11.x/octane)
Supercharges your application's performance by serving your application using high-powered application servers, including Swoole and RoadRunner. Octane boots your application once, keeps it in memory, and then feeds it requests at supersonic speeds.

### [Reverb](https://laravel.com/docs/11.x/reverb)
A first-party, high-performance WebSocket server for Laravel applications, enabling real-time communication between the server and the client.

## Monitoring & Debugging

### [Telescope](https://laravel.com/docs/11.x/telescope)
An elegant debug assistant for the Laravel framework. Telescope provides insight into the requests coming into your application, exceptions, log entries, database queries, queued jobs, mail, notifications, cache operations, scheduled tasks, variable dumps, and more.

### [Pulse](https://laravel.com/docs/11.x/pulse)
A real-time health and performance monitoring tool for your Laravel application. It allows you to identify bottlenecks like slow routes and jobs, and monitor server usage.

## Development Tools

### [Sail](https://laravel.com/docs/11.x/sail)
A light-weight command-line interface for interacting with Laravel's default Docker development environment. Sail provides a great starting point for building a Laravel application using PHP, MySQL, and Redis without requiring prior Docker experience.

### [Pint](https://laravel.com/docs/11.x/pint)
An opinionated PHP code style fixer for minimalists. Pint is built on top of PHP-CS-Fixer and ensures that your code style stays clean and consistent.

### [Dusk](https://laravel.com/docs/11.x/dusk)
Provides an expressive, easy-to-use browser automation and testing API. By default, Dusk does not require you to install JDK or Selenium on your local machine.

## Other Essential Packages

### [Cashier](https://laravel.com/docs/11.x/cashier)
Provides an expressive, fluent interface to Stripe's and Paddle's subscription billing services. It handles almost all of the boilerplate subscription billing code you are dreading writing.

### [Scout](https://laravel.com/docs/11.x/scout)
Provides a simple, driver-based solution for adding full-text search to your Eloquent models.

### [Pennant](https://laravel.com/docs/11.x/pennant)
A simple, lightweight library for managing feature flags in your application.

### [Folio](https://laravel.com/docs/11.x/folio)
A powerful page-based router designed to simplify routing in Laravel applications.

### [Prompts](https://laravel.com/docs/11.x/prompts)
A PHP package for adding beautiful and user-friendly forms to your command-line applications.
